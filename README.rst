pypi_release
~~~~~~~~~~~~

`setup.py release` command to support releasing your PyPI packages


Install
=======

Install from PyPI
-----------------

.. code-block:: bash

    $ pip install pypi_release

Install from Github repo
------------------------

.. code-block:: bash

    $ git clone https://github.com/laysakura/pypi_release.git
    $ cd pypi_release
    $ ./setup.py install
